# dwm

## File structure

* custom-scripts  
scripts for keybindings or other things  
    
* dotdwm  
equivalent of ~/.dwm folder, has autostart script  

* dwm  
the main dwm folder  

    * patches  
    * dwmblocks  
    the dwmblocks patch for the bar  

        * patches  
        * statusbar_scripts  
        the shell scripts for the statusbar
